# 耳聋基因检测系统

专业的耳聋基因检测与分析系统，基于PyQt6开发，为临床诊断提供精准支持。

## 功能特性

### 1. 登录界面
- 🔐 安全的用户认证系统
- 👥 多角色权限管理（管理员/医师/实验员）
- 🎨 现代化的界面设计
- 🔑 默认测试账号：admin/admin123, doctor/doctor123, tech/tech123

### 2. 主控台/首页
- 📊 实时数据看板（今日样本数、待审核报告数、已完成报告数、异常位点预警数）
- 🚀 快捷入口卡片
- 📋 最近活动记录
- ⏰ 自动刷新数据

### 3. 样本信息管理
- ➕ 单个新增/批量导入样本
- 🔍 搜索和筛选功能
- ✏️ 样本信息编辑
- 🗑️ 样本删除（带确认）
- 📤 导出样本列表

### 4. 基因数据解析（核心功能）
- 📁 支持Excel/CSV文件导入
- 🧬 集成现有基因分析算法
- 📈 实时解析进度显示
- 📋 位点展示表格
- 💾 解析结果保存
- 🔄 支持手动修正

### 5. 报告生成与预览
- 📄 多种报告模板（临床诊断/新生儿筛查/遗传咨询）
- 👁️ 实时HTML预览
- ✏️ 在线编辑功能
- 📝 添加备注
- 📑 PDF导出
- 🖨️ 打印支持
- ✅ 提交审核流程

### 6. 报告审核中心
- 📋 待审核/已审核/驳回报告分类
- ✅ 通过/驳回操作
- 💬 驳回原因填写
- 🔍 报告详情查看
- 📝 审核意见记录

### 7. 统计查询
- 📊 多维度筛选（时间、送检单位、基因类型、突变类型、检测结果）
- 📈 数据表格展示
- 📉 统计图表
- 📊 Excel导出
- 🖨️ 报表打印

### 8. 系统设置（管理员权限）
- 👥 用户管理（新增/编辑/删除）
- 📄 模板管理（自定义报告模板）
- 💾 数据库维护（备份/还原/清理日志）
- 📚 基因知识库更新
- 📋 操作日志查看

## 技术架构

### 后端技术
- **Python 3.8+**
- **SQLite** - 轻量级数据库
- **pandas** - 数据处理
- **openpyxl** - Excel文件处理
- **python-docx/docxtpl** - Word文档生成

### 前端技术
- **PyQt6** - GUI框架
- **PyQt6-WebEngine** - HTML预览支持
- **QSS** - 样式表

### 核心功能集成
- **基因分析算法** - 集成现有的deaf-gene报告生成功能
- **变异检测** - 支持4个常见耳聋基因（GJB2, GJB3, SLC26A4, MT-RNR1）
- **报告生成** - 自动生成Word和PDF格式报告

## 使用指南

### 基本工作流程

1. **登录系统**
   - 使用相应的账号登录
   - 不同角色看到的功能模块不同

2. **样本管理**
   - 添加样本信息（单个或批量）
   - 填写受检者基本信息、临床诊断等

3. **基因分析**
   - 导入检测数据文件（Excel格式）
   - 系统自动解析基因变异
   - 查看和确认分析结果

4. **报告生成**
   - 选择样本和报告模板
   - 预览报告内容
   - 编辑和添加备注
   - 生成PDF报告

5. **报告审核**
   - 医师审核待审核报告
   - 通过或驳回（填写原因）
   - 查看审核历史

6. **统计查询**
   - 按条件筛选数据
   - 查看统计图表
   - 导出统计报表

## 核心功能说明

### 基因数据解析
系统集成了现有的基因分析算法，支持：
- **GJB2基因** - 先天性重度或极重度感音神经性耳聋
- **GJB3基因** - 后天高频感音神经性耳聋
- **SLC26A4基因** - 大前庭水管综合征
- **MT-RNR1基因** - 药物敏感性耳聋

### 报告生成
- 自动分析基因变异结果
- 生成专业的检测报告
- 支持多种报告模板
- HTML实时预览
- PDF导出和打印

### 数据安全
- SQLite本地数据库
- 用户密码加密存储
- 操作日志记录
- 数据备份和还原

## 开发说明

### 扩展功能
1. **添加新的基因检测位点**
   - 修改 `config.py` 中的基因配置
   - 更新 `mutations.json` 变异数据库

2. **自定义报告模板**
   - 在系统设置中配置模板参数
   - 修改报告生成逻辑

3. **添加新的统计图表**
   - 集成matplotlib或其他图表库
   - 在统计查询界面添加图表展示

### 数据库结构
- `users` - 用户表
- `samples` - 样本表
- `gene_data` - 基因数据表
- `reports` - 报告表
- `audit_logs` - 审计日志表

## 注意事项

1. **首次运行**
   - 系统会自动创建数据库和默认用户
   - 建议首次登录后修改默认密码

2. **数据备份**
   - 定期在系统设置中备份数据库
   - 备份文件存储在 `data/backups/` 目录

3. **文件格式**
   - 样本导入支持 `.xlsx`, `.xls`, `.csv` 格式
   - 确保Excel文件包含必要的列名

4. **系统兼容性**
   - 支持Windows 7+
   - 支持主流Linux发行版
   - 支持macOS 10.14+
